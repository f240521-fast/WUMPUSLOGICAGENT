# WUMPUSLOGICAGENT
# Dynamic Wumpus Logic Agent (Web-Based AI Project)

## Overview
This project is a **Web-Based Dynamic Wumpus World Agent** that uses **Propositional Logic and Knowledge-Based Reasoning** to navigate a grid environment containing hidden hazards (Pits and a Wumpus).

The agent perceives its environment through **Breeze and Stench signals**, builds a **Knowledge Base (KB)**, and uses logical inference to decide safe moves.

---

## Features

- Dynamic grid size (user-defined Rows × Columns)
- Random placement of:
  - Pits
  - Wumpus
- Real-time percept generation:
  - Breeze (near pits)
  - Stench (near Wumpus)
- Knowledge Base (KB) using propositional logic
- TELL and ASK operations for reasoning
- Safe cell inference mechanism
- Resolution-based safety checking (basic implementation)
- Interactive web-based visualization
- Real-time inference step counter
- Agent movement visualization
